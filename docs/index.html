Dokumentasi Tesla Radix

Selamat datang di dokumentasi resmi Tesla Radix Bot v5.2. Sistem bot Minecraft paling canggih dengan arsitektur modular dan integrasi Telegram mendalam.

üöÄ Mengapa Tesla Radix?

Berbeda dengan bot konvensional, Tesla Radix menggunakan pendekatan Event-Driven. Ia memiliki "otak" (Core) yang terpisah dari "kemampuan" (Modules), membuatnya sangat ringan, stabil, dan mudah dikembangkan.

üì¶ Instalasi Cepat

Pastikan Anda sudah menginstall Node.js v18 ke atas.

git clone https://github.com/user/tesla-radix.git
cd tesla-radix
npm install
npm start

Jangan lupa konfigurasi file .env Anda sebelum menjalankan.

üß© Sistem Modul

Tesla Radix memuat kemampuan dari folder src/modules/. Berikut adalah modul standar yang tersedia:

  • Gameplay Mod: Menangani /mine, /build, dan auto-farming.
  • Radar Mod: Memindai entitas di sekitar bot (/scan).
  • System Mod: Menangani UI menu Telegram dan Start/Stop engine.
  • Moderasi Mod: Anti-link dan bad word filter.
